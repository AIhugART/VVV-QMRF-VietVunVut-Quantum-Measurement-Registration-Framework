## Scope

This file expands the compact Buddhist Epistemology system table into a strict line-by-line RCA concept system using the line-evidenced [RCA concept table](../documents/published_documents/The_Buddhist_Pramana_Epistemology_Logic_and_Language_RCA_Table.md) as source material.

## Source of truth policy

This file is the single source of truth for Buddhist Epistemology node and edge definitions used in RCA. Other BE node/edge tables are compact or derived references.

## RCA status labels (LLM-friendly)

The `RCA status` column is a reliability-and-provenance label for each concept or edge. It tells an LLM how the row should be interpreted relative to the cited source lines:

- `core synthesis`: synthesized from source evidence into the compact core system.
- `direct`: explicitly supported by cited source lines.
- `implied`: inferred from the source framework; use as a cautious structural relation, not a direct claim.
- `uncertain`: weak or incomplete source support; do not treat as settled.
- `listed only`: term appears in the source but is not explained enough for expansion.
- `OCR-normalized`: spelling was normalized from OCR/source noise; preserve the normalized form but verify if critical.
- `meta`: describes the article, method, or research context rather than a primary doctrine.
- `—`: no special RCA qualifier beyond the cited source line.

## Node code policy

- Core Buddhist Epistemology nodes use five-digit codes from N_BE_00001 to N_BE_00030.
- Line-by-line RCA nodes continue with N_BE_00031 and higher.
- Older shorter node-code forms are not used in this file.
- The first 30 rows preserve the compact system nodes; the RCA rows then preserve every source RCA row separately, including concepts already synthesized by the core rows.
