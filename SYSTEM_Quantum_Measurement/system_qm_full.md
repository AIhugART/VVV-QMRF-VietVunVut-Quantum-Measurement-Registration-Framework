## RCA status labels (LLM-friendly)

The `RCA status` column tells an LLM how strongly each node or edge should be trusted relative to the source table:

- `direct`: explicitly supported by the source row definition.
- `merged direct`: directly synthesized from both source traditions marked `S+J` in the source table.
- `partial`: source row is marked `[PARTIAL]`; use only within the verified scope.
- `extrapolated`: source row is marked `[NGOẠI SUY]`; do not treat as fully source-contained.
- `inferred`: source row is marked `[SUY DIỄN]`; use cautiously as an inferred concept.
- `implied`: relation is structurally implied by nearby definitions or category logic, but not stated as a standalone proposition.
- `uncertain`: relation has weak support; avoid using it for strong claims.
