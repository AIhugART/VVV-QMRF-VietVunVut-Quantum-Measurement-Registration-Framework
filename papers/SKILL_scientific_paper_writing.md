# SKILL: Scientific Paper Writing
> Rhetoric, RCA, and framing for research papers — optimized for revolutionary ideas.

---

## Core Axiom

**Revolutionary content demands conservative form.**
The more radical the claim, the more humble, precise, and structurally orthodox the language must be.
Never self-declare importance. Let data speak. Let others call it groundbreaking.

---

## 1. Root Cause Analysis (RCA) — Five Layers

| Layer | Question | Role |
|---|---|---|
| 1. Phenomenon | What is observed? | Empirical anchor / trigger |
| 2. Proximate Cause | What directly produces it? | Surface mechanism |
| 3. Underlying Mechanism | What produces the proximate cause? | Core contribution |
| 4. Root Cause / New Principle | What assumption must change? | Revolutionary claim — stated conservatively |
| 5. Generalization | Where else does this apply? | Scope of contribution |

**RCA validity checklist:**
- Falsifiable: generates predictions that could be wrong
- Parsimonious: fewest new assumptions necessary
- Predictive: anticipates unobserved phenomena
- Reproducible: survives independent replication
- Correspondent: recovers prior successful theory as limiting case

## 2. Framing Rules

### Rule 1 — Never self-label
- ❌ `"This paper fundamentally overturns QM..."`
- ✅ `"We propose an alternative formulation that may offer a more parsimonious account of [X]..."`

### Rule 2 — Anchor → Gap → Extend (never Reject → Replace)
```
"Building on [Prior Work], which established that [Z],
we note that this framework does not account for [gap].
Here we show that..."
```

### Rule 3 — Steel-man before diverging
1. Defend the prior framework explicitly
2. Identify one specific gap (not global failure)
3. Frame contribution as extension, not replacement

```
"The [framework] has proven remarkably successful in [X, Y, Z].
However, it encounters difficulty when applied to [specific case].
The present work does not seek to invalidate this framework
but rather examines whether [modified assumption] better accounts for [anomaly]."
```

### Rule 4 — Scope conservatively
Claim the minimum scope the data support. Let the community generalize.

---

## 3. Verb & Hedge Calibration

| Certainty | Context | Phrasing |
|---|---|---|
| High | Direct experimental result | `demonstrate, establish, show` |
| Moderate | Inference from data | `find, suggest, indicate` |
| Theoretical | New framework | `propose, hypothesize, posit` |
| Speculative | Future possibility | `raises the possibility that, may reflect` |

**Match linguistic certainty to evidential strength — no more, no less.**

---

## 4. Structural Templates

### Abstract (5-sentence formula)
```
[1] [X] is a central problem in [field].
[2] While [prior work] established [Z], [specific gap] remains unresolved.
[3] Here we [investigate / examine / develop] [approach].
[4] We find that [result — stated conservatively].
[5] This suggests [modest claim], with potential implications for [broader area].
```

### Introduction (paragraph sequence)
1. General context — why this problem matters
2. What is known — cite generously
3. Specific gap — narrow and precise, not global failure
4. Your approach — what you did
5. What you found + paper organization

### Discussion (sequence)
1. Restate main finding in plain language
2. Relate to prior work — convergence and divergence
3. Mechanistic interpretation (hedged)
4. Limitations — state them before the reviewer finds them
5. Generalization (careful scope)
6. Future directions

## 5. Vocabulary

### Avoid
| Word / Phrase | Replace With |
|---|---|
| revolutionary, groundbreaking, unprecedented | novel, previously unreported |
| proves (empirical claim) | demonstrates, provides evidence for |
| obviously, clearly | As shown in Figure X / Our data indicate |
| I believe, I argue | We propose / The data suggest |
| X is wrong, X fails | X does not account for / X encounters difficulty with |

### Use
| Function | Phrase |
|---|---|
| Acknowledge prior work | `"Building on the foundational work of [X]..."` |
| Signal gap | `"...an aspect that has not been systematically examined..."` |
| Introduce contribution | `"Here we provide, to our knowledge, the first..."` |
| Hedge interpretation | `"One interpretation consistent with these data is..."` |
| Preempt alternatives | `"Although we cannot exclude [X], [Y] argues against it..."` |
| Scope the claim | `"Within the limits of the present study..."` |
| Acknowledge limits | `"A limitation of the current approach is..."` |

## 6. Historical Anchor — Einstein 1905

Einstein did **not** write: *"Newton is wrong. Space and time are relative."*

He wrote: *"We will raise this conjecture to the status of a postulate..."*

| Device | Application |
|---|---|
| Grounded in accepted law | Opened by affirming Maxwell's equations |
| Conjecture → Postulate | Radical metaphysics framed as technical assumption |
| No ontological claim | Said nothing about what space/time "really are" |
| Correspondence preserved | Showed Newtonian mechanics as the low-velocity limit |
| Zero self-congratulation | Abstract contains no adjectives about the paper's importance |

## 7. LLM-Friendly Conventions

- Section numbers explicit: `1.`, `1.1`, `1.1.1`
- One claim per paragraph; main claim in first sentence
- Parallel grammar in all lists
- Metadata fields for machine parsing:

```yaml
title:
field:
contribution_type: [empirical | theoretical | methods | review]
main_claim:
contra_claim:
evidence_type: [experimental | computational | analytical | observational]
limitations:
scope: [narrow | field-wide | cross-disciplinary]
```

## 8. Quick Reference

| Principle | Rule |
|---|---|
| Revolutionary content, conservative form | Write as if extending, not overturning |
| Hedge proportionally | Certainty = evidence strength, nothing more |
| Steel-man first | Defend prior framework before identifying limits |
| Data leads, interpretation follows | Mark all interpretation with epistemic verbs |
| Scope conservatively | Claim minimum; let others generalize |
| Acknowledge limits actively | Find your weaknesses before the reviewer does |
| Never self-declare | Submit; let the community call it groundbreaking |
